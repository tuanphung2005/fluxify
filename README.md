## Architecture Charts

Architecture documentation is available as PlantUML source files in:

- `charts/index.md` for chart catalog and usage notes
- `charts/c4` for C4-style views
- `charts/uml` for UML domain and behavior views

### PlantUML Workspace Setup

- Install the recommended extension: `jebbs.plantuml`.
- Open any `.puml` file and run `PlantUML: Preview Current Diagram` from the Command Palette.
- Rendering is configured to use the public PlantUML server at `https://www.plantuml.com/plantuml`.
- Exports default to SVG in `charts/out`.
